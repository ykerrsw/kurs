\doxysubsection{Подробное описание}
Класс для обработки и записи ошибок. 

Класс \mbox{\hyperlink{classErrors}{Errors}} предоставляет функциональность для записи информации об ошибках в лог-\/файл, а также для управления именем файла лога.
